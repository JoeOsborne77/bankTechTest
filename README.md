# SETUP INSTRUCTIONS 

1. Fork/clone repo
 
2. Setup environment to use node latest version with;
#### $ nvm use node

3. Add jest to dependencies with;
#### $ npm add jest

4. Install dependencies with;
#### $ NPM install

5. Run tests with;
#### $ jest
